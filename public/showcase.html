🔄 상태 머신
draft → sent → viewed → signed → completed
draft: 작성완료 → [전송] 버튼
sent: 근로자에게 발송 → 근로자가 열람 시 자동 viewed
viewed: 근로자 검토 완료 → 서명 시 signed
signed: 서명 완료 → 사장님 [확정] → completed
completed: 계약 완료 배지 + PDF 다운로드
취소 경로: draft/sent/viewed → [취소] → cancelled
